# Students Performance in Exams: Prediction & Analysis

A data-driven study to analyze the influence of parental education, test preparation, and socio-economic factors on student academic scores, featuring a high-accuracy classification model.

## Project Overview
Can we predict if a student will pass or fail based on their background? This project explores a dataset of student scores in Math, Reading, and Writing to identify key success factors and build a predictive model.

## Tech Stack
* **Language:** Python
* **Machine Learning:** Logistic Regression
* **Preprocessing:** Standard Scaling, Label Encoding.
* **Libraries:** Pandas, Seaborn, Matplotlib, Scikit-learn.

## Technical Implementation
- **Exploratory Data Analysis (EDA):** Visualized the impact of "test preparation courses" and "lunch type" on overall performance.
- **Target Engineering:** Created a binary classification target (Pass/Fail) based on weighted average scores.
- **Model Training:** Implemented Logistic Regression after normalizing numerical features.
- **Evaluation:** Achieved a stellar **97% accuracy** with balanced precision and recall.

## Key Insights
- Students who completed test preparation courses scored significantly higher.
- Parental education level has a positive correlation with student success rates.
- Financial factors (represented by lunch type) showed a visible impact on performance.
